インターフェース設計 / Interface Design

      

      

        

          

            
コンポーネント役割実装CGO

            
              

                Annotator
                形態素解析インターフェース
                interface{ Annotate(string) ([]Token, error) }
                —
              

              

                DictAnnotator
                組み込み辞書バックエンド
                3,000語 greedy longest-match trie scan
                不要
              

              

                MeCabAnnotator
                MeCab IPC アダプター
                subprocess stdin/stdout + IPAdic parser
                オプション
              

              

                Middleware
                HTTP レスポンス変換
                responseCapture → DOM walk → ruby inject
                不要
              

              

                InjectFurigana
                HTML変換関数 (スタンドアロン)
                golang.org/x/net/html parser + tree mutator
                不要
